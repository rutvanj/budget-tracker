# Budget Tracker (Python)

Budget Tracker is a simple command-line application built with Python that helps users track their daily expenses and calculate total spending.

## Features

- Add and store expenses
- View all recorded expenses
- Calculate total spending
- Data saved in a text file for persistence

## Tech Stack

- Python
- File Handling
